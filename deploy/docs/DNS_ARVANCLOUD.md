## مراحل تنظیم در پنل ابرآروان

### مرحله 1: ورود به پنل
1. وارد https://panel.arvancloud.ir شوید
2. به بخش **Cloud DNS** بروید
3. دامنه `adonis.gr` را انتخاب کنید

### مرحله 2: بکاپ از رکوردهای قبلی
⚠️ **مهم:** قبل از هر تغییری:
1. روی **Export** کلیک کنید
2. فایل JSON یا CSV رکوردهای فعلی را دانلود کنید
3. در جای امن نگهداری کنید

### مرحله 3: حذف رکوردهای قبلی A/AAAA
1. رکورد A فعلی برای `@` را حذف کنید (IP قبلی: 213.158.90.235)
2. رکورد A یا CNAME فعلی برای `www` را حذف کنید

### مرحله 4: اضافه کردن رکوردهای جدید
1. روی **Add Record** کلیک کنید
2. رکوردهای جدول بالا را وارد کنید

### مرحله 5: تنظیم TTL
- **در زمان مهاجرت:** TTL = 300 (5 دقیقه)
- **بعد از اطمینان:** TTL = 3600 (1 ساعت)

## تنظیمات CDN/Proxy ابرآروان

### قبل از SSL
1. **Proxy را OFF کنید** (Cloud icon خاکستری)
2. این اجازه می‌دهد Certbot مستقیم به سرور دسترسی داشته باشد

### بعد از SSL
اگر می‌خواهید از CDN ابرآروان استفاده کنید:
1. **Proxy را ON کنید** (Cloud icon نارنجی)
2. در تنظیمات SSL ابرآروان: **Full (Strict)** انتخاب کنید
3. Origin Certificate نیاز نیست چون Let's Encrypt داریم

## تست DNS

### از سرور:
```bash
dig +short adonis.gr A
# باید نمایش دهد: 116.203.240.156

dig +short www.adonis.gr A
# باید نمایش دهد: 116.203.240.156
```

### آنلاین:
- https://dnschecker.org
- https://whatsmydns.net

## بعد از تنظیم DNS

1. صبر کنید تا DNS propagate شود (معمولاً 5-30 دقیقه)
2. تست کنید:
   ```bash
   curl -I http://adonis.gr
   ```
3. SSL را فعال کنید:
   ```bash
   cd /root/adonis_site
   ./deploy/scripts/setup-ssl.sh admin@adonis.gr
   ```

---

## نکات مهم

1. **Propagation Time:** تغییرات DNS ممکن است 24-48 ساعت طول بکشد، ولی معمولاً سریع‌تر است
2. **Cache:** مرورگر و ISP ممکن است DNS قدیمی را cache کرده باشند
3. **نسخه فارسی:** سایت فارسی در مسیر `/fa-new/` قابل دسترس است
4. **ریدایرکت:** www.adonis.gr به adonis.gr ریدایرکت می‌شود
